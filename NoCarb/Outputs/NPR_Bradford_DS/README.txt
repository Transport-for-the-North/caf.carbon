##################################################################################################
NoCarb Outputs (created by running the rail_calculations.py script inside caf.carbon)
##################################################################################################


########################################################
annual_summary_emissions
########################################################
Provides emissions (kg/CO2e) aggregated to year and vehicle type. An interpolated version for all years between 2018 and 2052 
and another version which includes 2018, 2042 and 2052 only.


########################################################
annual_zonal_emissions
########################################################
Provides emissions (kg/CO2e) for each year and each zone. An interpolated version for all years between 2018 and 2052 
and another version which includes 2018, 2042 and 2052 only.


########################################################
scheme_emissions
########################################################
Provides emissions (kg/CO2e) for each service listed in the scheme, includes the year and time period. An interpolated version for all years 
between 2018 and 2052 and another version which includes 2018, 2042 and 2052 only.
